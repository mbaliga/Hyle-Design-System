## The core law

<div style={{ fontFamily: "'Archivo Variable', system-ui, sans-serif", fontWeight: 600, fontSize: 'clamp(22px, 4vw, 40px)', lineHeight: 1.15, color: '#ece8e4', borderLeft: '3px solid #8e7bff', padding: '4px 0 4px 20px', margin: '8px 0 24px' }}>
  State is SHOWN by material behavior,<br/>never SAID by language.
</div>

No status words. No spinners. Hyle tells the truth about one thing — *computation* — and tells it through material: **light only where thinking happens, motion only when there is real state, mass proportional to depth of thought.** Honesty is relocated from *substance* to *behavior*.

## The material stack

Every surface in Hyle is one of these, layered:

- **The Field** — the living layer of monumental forms in coloured haze (the Form-World engine). Matte and near-still at rest; it moves only when there is real computation. This is where state is *shown*. → *Material / Field*
- **The Pane** — frosted glass floating over the Field, carrying the readable controls. Glass means legibility; it stays calm and still. → *Material / Pane*
- **The Tactile Kit** — where the interface becomes hardware: knobs, faders, meters, a scroll wheel — "soft brutalism", *controls you want to touch*, in honest materials (machined metal, glass, leather, concrete). → *Material / Tactile Kit*

The single violet accent (`#8E7BFF`) is the only colour spent freely — and it is **always the user's to choose** (the whole system re-themes from one picked colour). Everything else is earned.

## The lineage (why it coheres)

Hyle draws on lineages that all converge on one move — *meaning is carried by matter behaving, not by language describing*:

**Hylomorphism** (form emerges from matter) · **Calm technology** (Weiser/Case — "technology can communicate but doesn't need to speak") · **Affordances** (Gibson — a surface specifies its own use) · **Truth to materials** (Bauhaus, Rams, brutalism) · **Material perception** (Fleming — the eye reads correct light/refraction as *real*) · **Cymatics & ferrofluid** (a field/frequency *makes* the pattern) · **The mathematical sublime** (Kant — vastness that calms: *"calm because massive"*).
