# Hyle

> *Hyle* (ὕλη) — Aristotle's word for matter: pure potentiality awaiting form.

Hyle is a cross-platform design system built on a single law:

## The core law

> **State is SHOWN by material behavior, never SAID by language.**

No status words. No spinners. The interface tells the truth about one thing —
*computation* — and tells it through material: light only where thinking
happens, motion only when there is real state, mass proportional to depth of
thought. Honesty is relocated from *substance* to *behavior*.

## The two layers

Hyle is built as a material stack:

1. **The Field** — a living layer of monumental forms in coloured haze
   (the Form-World WebGL engine). Matte and near-still at rest; it moves only
   when there is real computation. This is where state is *shown*.
2. **The Pane** — frosted glass that floats over the Field carrying the
   readable controls. Glass means legibility; it stays calm and still, animating
   only to enter or leave.

Controls on the glass — **chips**, sliders, micro-labels — are quiet hairline
outlines until pressed, when they invert to solid ink. The single violet accent
(`#8E7BFF`) is the only colour spent freely; everything else is earned.

## Architecture

```
tokens/*.json        Style Dictionary        build/
(W3C DTCG)      ──►   build-tokens.js   ──►   web/      tokens.css · scss · js
                                              android/  colors.xml · dimens.xml
                                              ios/      Tokens.swift

src/components/   ──► Lit web components ──►   <hy-field> <hy-pane> <hy-chip> …
field/            ──► Form-World engine  ──►   the living material behind the pane
```

One token definition compiles to every platform — Android, web, iOS — because
Hyle is meant to live everywhere, not just the browser.
